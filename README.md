# Zenergy

Zenergy is a social wellness web application built with React and Firebase that allows users to share vibes, connect with others, and track positive interactions.

## Features
- User authentication with Firebase
- Post and share vibes
- Social interaction between users
- Modern UI built with React and Vite

## Run Locally

### Prerequisites
- Node.js installed

### Install dependencies
npm install

### Run the app
npm run dev

The app will run on:
http://localhost:3000
